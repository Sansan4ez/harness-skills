Repo Harness Kit v1.0.0
=======================

Compatibility Impact
--------------------

- Release type: `major`
- Target-repo action: `update`
- Migration style: no path migration inside v1.0.0 itself; this release freezes the v1 surface.

What Is Included
----------------

- Stable module entrypoints under `skills/<skill>/scripts/**`
- Rollback-safe `install|update|check` flow with `harness/kit-lock.yaml`
- Docs, traceability, and observability harness modules
- Release-matrix validation across golden fixtures
- Bootstrap flow for new repos
- Agent skill packaging for the three harness modules
- Docs-only service kinds for `typescript`, `javascript`, `go`, and `rust`

Upgrade Notes
-------------

1. Bootstrap a new repo with:

```bash
uv run python skills/<skill>/scripts/harness/bootstrap.py --repo-root <repo>
```

2. Upgrade an existing repo module-by-module with:

```bash
uv run python skills/<skill>/scripts/harness/harness.py --repo-root <repo> --mode update --module <module> --kit-version 1.0.0
```

3. Re-run verification:

```bash
uv run pytest tests/harness -q
uv run python scripts/harness/release_matrix.py --kit-version 1.0.0
```
