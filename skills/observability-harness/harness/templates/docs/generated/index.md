Generated Inventories
=====================

Purpose
-------

Generated inventories provide a mechanical view of repository surfaces (docs, contracts, env vars, and observability assets). They help keep implementation aligned with docs and specs.

Scope
-----

- Service inventory -> `docs/generated/service-inventory.md`
- Doc inventory -> `docs/generated/doc-inventory.md`
- Doc gardening report -> `docs/generated/doc-gardening-report.md`
- Env var inventory -> `docs/generated/env-matrix.md`

Notes
-----

- This index is updated by the `docs-harness` skill.
