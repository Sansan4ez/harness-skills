C4 Architecture Spine
=====================

Purpose
-------

This folder provides the C4-based architecture spine for the repository:

- L1: system context
- L2: containers (services)
- L3: components (per service)

Artifacts
---------

- L1 system context -> `docs/architecture/c4/l1-system-context.md`
- L2 containers -> `docs/architecture/c4/l2-containers.md`
- L3 components (per service) -> `docs/architecture/c4/services/<service>/l3-components.md`
